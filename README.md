# Bundl

A rails web application where users can create, view, update and delete events. You can view the live [demo here](https://www.youtube.com/watch?v=pnDAT-erJuM).

## Usage

Signup or login to CRUD events.

## Built Using

- Ruby on Rails
- Devise
- PostgreSQL
